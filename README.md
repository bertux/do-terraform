# do-terraform
Automatic provisioning by Terraform on Digital Ocean

## install Go
```
sudo snap install go --classic
echo 'export GOPATH=$HOME' >> .bashrc
```
## install Docker
```
curl -fsSL https://get.docker.com -o get-docker.sh
sh get-docker.sh
```
## install Parity
```
bash <(curl https://get.parity.io -L)
```
